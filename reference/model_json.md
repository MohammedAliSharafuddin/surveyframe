# Serialise a model specification to JSON

Serialise a model specification to JSON

## Usage

``` r
model_json(model, pretty = TRUE)
```

## Arguments

- model:

  An
  [`sf_model()`](https://mohammedalisharafuddin.github.io/surveyframe/reference/sf_model.md)
  object.

- pretty:

  Logical. Whether to pretty-print the JSON.

## Value

A JSON string.
